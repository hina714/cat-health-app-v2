# 猫登録機能 仕様書

## 概要

ログイン中のユーザーが、飼っている猫のプロフィールを登録・閲覧・削除できる機能。

---

## データ仕様

### cats テーブル

| カラム | 型 | 必須 | 説明 |
|---|---|---|---|
| `id` | UUID | ✓ | 主キー（自動生成） |
| `user_id` | UUID | ✓ | 登録者（`users.id` への参照） |
| `name` | TEXT | ✓ | 猫の名前 |
| `icon_data` | TEXT | | アイコン画像（base64） |
| `breed` | TEXT | | 品種 |
| `birthdate` | DATE | | 誕生日 |
| `created_at` | TIMESTAMPTZ | | 登録日時（自動セット） |

---

## API仕様

### GET /api/cats
ログイン中のユーザーの猫一覧を返す。

- 認証: 必須（未ログインは 401）
- レスポンス: `{ ok: true, cats: [...] }`

### POST /api/cats
猫を新規登録する。

- 認証: 必須（未ログインは 401）
- リクエスト: `{ name, icon_data?, breed?, birthdate? }`
- バリデーション: `name` が空の場合は 400
- レスポンス: `{ ok: true, cat: {...} }`（201）

### DELETE /api/cats/:id
指定した猫を削除する。

- 認証: 必須（未ログインは 401）
- 制約: 自分の猫のみ削除可能（他人のIDを指定しても 404）
- レスポンス: `{ ok: true }`

---

## 画面仕様

### 猫登録画面（/cats/new）

| 項目 | 種類 | 必須 | 備考 |
|---|---|---|---|
| アイコン写真 | ファイル選択 | | 選択後にトリミング・ズーム調整が可能 |
| 名前 | テキスト入力 | ✓ | |
| 品種 | テキスト入力 | | |
| 誕生日 | 日付入力 | | |

**写真編集の仕様**
- 写真を選択するとエディタが表示される
- ドラッグで位置を調整できる
- スライダーでズームを調整できる（1〜3倍）
- 「写真を変更する」で再選択できる
- 登録時に編集内容を JPEG（品質 85%）で確定し、base64 に変換して保存する
- 上限サイズ: 10MB

**登録後の動作**
- 成功したらトップページ（`/`）へリダイレクト

### トップページ（/）

**猫が1件以上登録されている場合**
- 「うちの子」セクションに猫のアイコンと名前を一覧表示
- アイコン写真がない場合は 🐱 を表示
- アイコンにホバーすると削除ボタン（✕）が表示される

**削除の動作**
- ✕ ボタンを押すと確認ダイアログを表示（「元に戻せません」と明記）
- OK で削除、キャンセルで何もしない

**猫が0件の場合**
- 「猫を登録する」カードを表示し、`/cats/new` へのリンクとする
